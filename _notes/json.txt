------------------------------------
JSON
------------------------------------

Json - формат для представления значений и объектов структурирования данных на основе синтаксиса объекта JavaScript. Используется для передачи данных. Может быть в двух состояниях: Json - строка, Json - объект.
JS предоставляет глобальный объект Json, который имеет методы для преобразования :

Методы:
------------------------------------
-JSON.stringify({object}) - сериализация (объект => строка)
-JSON.pars('string') - десериализация (строка => объект)

Типы данных JSON: 
- object
- array
- string
- number
- bullean
- null

Очень строг на синтаксические ошибки и ключом свойства JSON объекта всегда является строка.

https://jsonformatter.org - проверка на ошибки
