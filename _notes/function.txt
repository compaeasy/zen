Вызов функции это блииин () круглые скобки после имени функции!!!

Call function Это функция, которая передается в качестве аргумента в другую функцию

Arrow function - это стрелочная функция, функция, которая была добавлена в спецификацию JavaScript для удобства создания call back функции

() => {}

() - Область обьявления параметров функции, при объявлении параметра функции значение его undefined

Параметр функции - это локальная переменаая функции, то есть переменная, которая доступна только внутри этой функции

Аргумент функции - это значение, которое передается при вызове функции в круглые скобки: имяФункции(аргумент), то есть значение, которое теперь будет присвоено абстрактному аргументу, который был объявлен при определении функции

{} - это тело функции, то есть блок в котором мы можем написать какие-либо инструкции

return - это оператор, который указывает на то что мы хотим, чтобы возвращала функция. При вызове функции она может вернуть что-то один раз за один вызов. (второй вызов второй возврат, третий вызов третий возврат) .Что она будет возвращать, это оказывает оператор return. Код под оператором return не доступен, чтобы возврат был доступен, он должен быть указан на одной строке с оператором return!!!

Правильно: 
return 5 + 5 

Не правильно: 
return 
5 + 5
